# AMPS (Attendance and Performance Management System)

AMPS is a full-stack web application designed to manage student attendance and academic performance in colleges.

## Features
- Student and Admin authentication
- Attendance tracking (subject-wise, date-wise)
- Performance management (marks, internal exams)
- Dashboard with insights

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB Atlas

## Status
In Development (MVP Phase)

## Goal
To build a scalable system that simplifies attendance tracking and performance monitoring in educational institutions.
